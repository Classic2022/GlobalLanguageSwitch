Urban Reparaturen - Hostinger Deployment Package

This folder contains all the files you need to deploy your Urban Reparaturen website to Hostinger.

Included Files

hostinger_upload/ ├── .htaccess # Server configuration for routing, caching, etc. ├── form-handler.php # PHP script to handle contact form submissions ├── robots.txt # Instructions for search engine crawlers ├── sitemap.xml # XML sitemap for better SEO └── index.html # This instruction file

Deployment Instructions

  1. Run npm run build locally to generate your static site files
  2. Log into your Hostinger control panel
  3. Navigate to File Manager (or use FTP)
  4. Go to your domain's public_html folder
  5. Upload all the files from this folder to the public_html directory
  6. Upload all files from your local dist/public directory to the same location
  7. Ensure that you've replaced this index.html with the one from your build

Contact Form Setup

The form-handler.php file is configured to:

  • Receive form submissions from your contact form
  • Process and validate the data
  • Send an email notification to info@urban-r.de
  • Return a JSON response that your form can display to users
Note: If your contact form isn't working after deployment, make sure that:
  • PHP is enabled on your Hostinger account (it should be by default)
  • Your form is correctly submitting to /form-handler.php
  • The email address in the PHP file is correct

Final Testing

After uploading all files, make sure to test:

  • Navigation between all pages
  • The contact form submission process
  • Mobile responsiveness
  • That the favicon appears correctly
